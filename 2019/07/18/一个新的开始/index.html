一个新的开始

发表于 2019-07-18 | 分类于 生活 | | 阅读次数:

  7.16日，一个不一样的日子，因为今天是我作为实习生入职新蛋的第一天。带着激动的心情早上还下着小雨，第一天竟然还因为走错路而挤了一个早高峰的地铁。入职第一天肯定是不会直接参与项目或者办公什么的啦，而是对我们进行了一天的公司文化的熏陶。一开始当然是每个人做一个简单的自我介绍，做完自我介绍后进行了一个记忆力的小游戏，而我也恰恰就是两个游戏输家的一个，所以在中午也唱了我比较喜欢的一首歌《小幸运》。然后就开始了公司的介绍，从招聘、薪资、绩效、安全，再到部门、公司业务等讲了公司的整体架构，而我也带着100分的精神一直听到结束，最后给我安排到了一个做仓库业务的部门 OZZO 部，并且安排了一位部门内的导师，而在今天(7.17)导师也很友好的了解了一下我的情况并且告诉了我接下来应该做什么以及应该侧重什么，让我在近期也对该看什么有了一定的方向。第一天，很激动的开始也很快的结束了，我感受到的是大家的友好、热情以及对我们到来真正的开心，所以在接下来的日子我也需要加油啦~
  明天（7.18）就是正式开始培训的第一天，培训内容大致是 数据库 -> 后端 -> 前端 -> 测试这样的课程安排，我觉得我更应该打起100分的精神去听，课下除了做出布置的内容也应该去做一些拓展和思考，这样才能更加理解以便在后续再把理论变成实践，这可已经不是学校理论的背背背和仅仅是简单的写个Demo的事情了，而是需要付诸努力去考虑它的应用场景以及实践，这才可以在项目大作业中头脑清晰、比较轻松的完成。不管怎样，现在都应该慢慢去调整自己了，学校是学习的个人学习的地方，而公司是团队协作的地方，怎样做好自己，和团队好好沟通是我接下来应该去学习的东西，这样才能产生自己的价值，为公司创造价值，慢慢去寻找方式吧，该学习的地方还很多！

为第一家公司的入职做好准备~
